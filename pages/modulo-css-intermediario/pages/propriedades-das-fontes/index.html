Propriedades das fontes

(Obs.: Todos códigos citados neste arquivo html estão sendo demonstrados no arquivo style.css dessa pasta)

É possível mudar a fonte usando o google fonts e o site dafont

Obs.: Existem fontes pagas também

∘ Passo a passo de como adicionar uma fonte do google:

1° Passo: Pesquisar google fonts e selecionar as grossuras da fonte desejada

2° Passo: Copiar e colar o @import da fonte para a aba style.css

3° Passo: Copiar e colar a especificidade da fonte usando o código font-family: nome da especificidade;

• Exemplo:

@import url('https://fonts.googleapis.com/css2?family=Open+Sans:ital,wght@0,300;0,400;0,600;0,700;1,300;1,400;1,500;1,600;1,700&display=swap');

html {
font-family: 'Open Sans', sans-serif;
font-weight: 500;
}

∘ Passo a passo de como adicionar uma fonte do site DaFont:

1° Passo: Pesquisar DaFont, escolher a fonte, olhar se tem todos os grifos e baixar a fonte

2° Passo: Criar uma pasta com o nome fonts dentro da pasta raiz da página html

3° Passo: Copiar e colar a fonte baixada para a pasta fonts

4° Passo: Seguir o eemplo a baixo

• Exemplo:

Usar o seguite código a baixo na aba style.css

@font-face {
font-family: 'inventar o nome da fonte' ;
src: url(./ caminho da fonte);
}

h1 {
font-family: 'nome da fonte inventada';
font-weight: grossura da fonte (100 a 900);
}
