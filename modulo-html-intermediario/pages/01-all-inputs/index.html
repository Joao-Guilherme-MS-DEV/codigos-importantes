Formulário com GET

O método GET passará os dados do formulário com os parâmetros na URL, como quando realizamos uma busca. NÃO UTILIZE O GET PARA INPUTS COM INFORMAÇÕES SENSÍVEIS COMO SENHAS, pois elas aparecerão na URL.

Formulário com POST

O método POST passará os dados do formulário por baixo dos panos. Utilizamos para tratar dados sensíveis como senhas e e-mails
